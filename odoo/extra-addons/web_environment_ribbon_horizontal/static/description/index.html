Web Environment Ribbon Horizontal

Description

This module is configured to identify a test environment by placing a customizable red ribbon at the top of each page, emphasizing the test environment's database name.

Functionality

A red ribbon is added on top of every page in the test environment as per the functionality.

Go to Settings - - > Technical - - > Parameters - - > System Parameters where the parameters such as ribbon.background color, ribbon.color and ribbon.name is set, which can be altered based on the requirement.
